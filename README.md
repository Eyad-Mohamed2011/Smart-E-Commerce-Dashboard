# 🚀 لوحة تحكم المتجر الذكي (Smart E-Commerce Dashboard)

مشروع لوحة تحكم متطورة ومستجيبة (Responsive) تم بناؤها باستخدام **React.js** ومكتبة **React-Bootstrap** لإدارة مخزون المنتجات بكفاءة. يوضح هذا المشروع استخدام أنماط React المتقدمة وتحسين الأداء.

## ✨ المميزات الرئيسية
- **إدارة الحالة الشاملة (Global State):** تم التنفيذ باستخدام `useReducer` و `Context API`.
- **تحسين الأداء (Performance):** استخدام `useMemo` لفلترة المنتجات في الوقت الفعلي دون إعادة تحميل غير ضرورية.
- **التركيز التلقائي (Auto-Focus):** استخدام `useRef` لتركيز المؤشر على شريط البحث فور تشغيل التطبيق.
- **محاكاة البيانات (Mock API):** محاكاة جلب البيانات باستخدام `useEffect` مع تأخير زمني (Delay).
- **Hooks مخصصة:** فصل منطق العمل (Logic) باستخدام `useForm` و `useProducts`.
- **Portals:** استخدام `ReactDOM.createPortal` لعرض النوافذ المنبثقة (Modals) بطريقة احترافية.

## 🛠️ التقنيات المستخدمة
- **Frontend:** React.js
- **Styling:** React-Bootstrap & CSS3
- **State Management:** Context API + useReducer
